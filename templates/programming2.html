れきぷら

プロジェクトの目的

本プロジェクトは、ICT社会応用論dの4Daysチーム開発の成果物であり、VectorDBやRAGを用いたChatGPTを利用した動的なWebアプリを、5人で4日間で開発した。

作成方法

Qdrant RAG Django Python JavaScript HTML CSS

詳細

歴史を気軽に学べるWebアプリとして、歴史上の人物を入力するとその人物ゆかりの地と詳細が表示される。VectorDBとしてQdrantを用いてRAGを行い、Wikipediaなどの情報を得た歴史に強いChatGPTを作成し、活用している。ChatGPTを用いた「れきしはかせ」への質問機能も備えている。

担当箇所

アプリ画面のフロントエンドを担当。デザイン班の和風な案をもとに、Bootstrapを用いてHTML/CSSを作成した。
